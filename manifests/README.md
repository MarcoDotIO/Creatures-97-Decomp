# Manifests

This directory contains cross-cutting inventories and mapping files.

Key files:

- `reconstruction_naming_map.tsv`
  Canonical original-symbol to recovered-name/status mapping.
- `decomp_scope.tsv`
  Binary and subsystem scope tracking.
- `creatures_binaries.tsv`
  Inventory of the executable surface being reconstructed.
- `creatures_install_tree.tsv`
  Install-tree and file-layout references.
- `egg_disk_inventory.tsv`
  Supporting binary/media inventory data.

If you need a canonical answer for "what is this function called now?" or
"which binary owns this work?", start here.
